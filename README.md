# microcodes-library
This my personal library with footprints for Eagle v8, which I've been updating for the past few years.

__Contains the following__

* TP/TC/AP/LTC 4056 PMIC
* 4953 mosfet
* 9926A mosfet
* MT3540 SOT23-6L
* AOD4185
* ATSAMD21G18A
* CN3761
* CN3791
* SDM1U20CSP Schottky 
* E19-868M20S / E19-433M20S / E19-915M20S
* ESP32-PICO-D4
* FP6276A
* FM5324B 
* MCP1702
* MT3420
* SMA3109
* STM32F401RCT6
* Sumida P0762NL
* Switch Slide SPDT 0.5A 50VDC
* SY8083P
* TC7662B/TC7660S
* TPA3118D2DAPR
* misc
